# Advice Generator App

A responsive advice generator web application built with HTML, CSS JavaScript & Fetch API.

## Project Overview

This is a simple, modern advice generator that displays random pieces of advice. Users can click the dice button to generate new advice calling fetch API.

## Features

- Responsive design (mobile & desktop)
- Clean, minimalist UI
- Click to generate new advice
- Smooth animations and hover effects

## Built With

- HTML5
- CSS3 (internal styles)
- JavaScript
- Google Fonts (Manrope)

## Getting Started

1. Open `index.html` in your browser
2. Click the dice button to generate new advice

## Design Details

- Colors: Dark blue theme with green accents
- Fonts: Manrope (weight 800)
- Breakpoints: Mobile (375px) and Desktop (1440px)
- Dividers: Responsive SVG patterns
